🚌 College Bus Tracking System

A real-time college bus tracking system built using Node.js, Socket.IO, and Leaflet Maps that allows students to track buses live, drivers to send GPS locations, and administrators to manage buses and routes.
This project demonstrates a fleet management and live location tracking system similar to apps used by transport companies.

🛠 Tech Stack

Frontend:
HTML5
CSS3
JavaScript
Leaflet.js (Map rendering) 

Backend:
Node.js
Express.js
Socket.IO

Database:
Local JSON files (for demo)
Can be upgraded to MongoDB
